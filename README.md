# TechnicalSolutions - Vercel Version

Modern business website built with:

- Next.js
- TypeScript
- Tailwind CSS
- Lucide React icons
- Vercel-ready deployment

## Run locally

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Deploy to Vercel

1. Push this project to GitHub.
2. Go to Vercel.
3. Click Add New Project.
4. Import the GitHub repository.
5. Use the default Next.js settings.
6. Deploy.
